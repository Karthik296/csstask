SELECTORS

  1. Simple
    • Element:elects elements by tag name, e.g., (p) selects all paragraphs.
    • Class(.):Selects elements with a specific class attribute.
    • ID(#):Selects a unique element by its ID.Should be one per page.
    • Grouping(,):Groups multiple selectors to apply the same styles.
    • Universal(*):Selects all elements. Useful for resetting styles globally.
  2. Combinator
  3. Attribute
  4. Pseudo classes
  5. Pseudo elements
